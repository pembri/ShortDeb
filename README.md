# ShortDeb - Chromium Shortcut to .deb Package Creator

**ShortDeb** is a tool that allows Linux users to create shortcuts for Chromium-based browser applications (such as Chromium-based web apps) and package them into `.deb` format, the installation package format for Debian-based Linux distributions (e.g., Debian). With this tool, users can easily create and distribute shortcuts to their desktop or application menu.

## Key Features

- **Create Chromium Application Shortcuts**: Generate shortcuts from existing Chromium-based apps or websites by entering a URL or selecting an already installed Chromium app.
- **Customize Icons and Names**: Add custom icons and give your shortcut a name.
- **Automatic `.deb` Package Creation**: Convert your shortcut into a `.deb` package, ready to be installed on Debian-based Linux distributions.
- **Compatibility with Different Chromium Versions**: Supports various versions of Chromium and Chromium-based apps (Chromium).
- **User-Friendly Interface**: A simple and intuitive graphical user interface (GUI) to make shortcut creation easy.

## How to Use

### Installation

1. **Download the Latest Release**  
   Go to the [Releases](https://github.com/username/ShortDeb/releases) page and download the latest release package (e.g., `ShortDeb-v1.0.0.deb`).

2. **Install the `.deb` Package using `dpkg`**  
   Once you've downloaded the `.deb` file, install the package using the following command:
   ```bash
   sudo dpkg -i ShortDeb-v1.0.0.deb
3. **Install the `.deb` Package using `gdebi`**  
   Once you've downloaded the `.deb` file, install the package using the following command:
   ```bash
   sudo gdebi ShortDeb-v1.0.0.deb
